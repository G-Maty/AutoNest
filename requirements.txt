# AutoNest v3.0 - フォルダ・ファイル自動作成ツール
# 実行用依存関係: 標準ライブラリのみ使用

# === 使用している標準ライブラリ ===
# tkinter - GUI (Python標準ライブラリ)
# tkinter.filedialog - ファイル選択ダイアログ
# tkinter.messagebox - メッセージボックス  
# tkinter.ttk - モダンなウィジェット
# os - ファイル・フォルダ操作
# sys - システム関連
# json - 設定ファイル読み込み
# shutil - ファイルコピー操作
# time - 時間処理
# pathlib - パス操作

# === 外部パッケージ ===
# 外部パッケージは使用していません

# === 実行環境要件 ===  
# Python 3.7以上
# tkinter（通常はPythonと一緒にインストール済み）
